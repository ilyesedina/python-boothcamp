# Band Name Generator

This is a beginner-friendly Python project that generates a band name based on user input.

## How it works

1. The program welcomes the user.
2. It asks for the name of the city where the user grew up.
3. It asks for the user's pet's name.
4. It combines these two inputs to suggest a band name.

## Example Output

Welcome to the Band Name Generator. 
What's the name of the city you grew up in?
**London** 
What's your pet's name? 
**Fluffy** 

Your band name is 'London Fluffy'!

## How to run

1. Make sure you have Python installed.
2. Run the script using a Python interpreter.
3. Follow the prompts to generate your band name.
